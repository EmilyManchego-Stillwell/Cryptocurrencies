# Cryptocurrencies
## Overview:
- Purpose:
    - The purpose of this project is to exhibit the different cryptocurrencies that are currently trading on the trading market, and how they can be classified. We determined how to group and classify the different cryptocurrencies being traded by using unsupervised machine learning.
